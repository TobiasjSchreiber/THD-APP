# THD App – UI/UX Redesign (Studienprojekt)

Dieses Projekt entstand im Rahmen des Moduls **Branded Interaction**. Ziel war die Entwicklung eines interaktiven High-Fidelity-Prototyps basierend auf dem **Human-Centred-Design (HCD) Prozess**.

## 🌐 Live-Demo & Prototyp
Die funktionale Web-App ist hier verfügbar:
👉 **[THD App Live-Link](https://tobiasjschreiber.github.io/THD-APP/)**

## 🎯 Projektziel & Fokus
Der Schwerpunkt liegt auf einer hochwertigen **User Experience (UX)** und einem begeisternden **Visual Design**. 
* **HCD-Ansatz:** Die App wurde schrittweise entwickelt, um die Bedürfnisse der Studierenden in den Mittelpunkt zu stellen.
* **Design-Fokus:** Modernisierung der THD-App durch ein intuitives und flüssiges Interface.

## ✨ Kern-Features (UX/UI)
* **Personalisiertes Dashboard:** Widgets einfach per Drag & Drop anordnen.
* **Native App-Experience:** Wischgesten, Fullscreen-Modus und flüssige Animationen.
* **Responsive Design:** Optimiert für Smartphones und mit eigenem Layout für Tablets.
* **Modernes Theming:** Unterstützung von Light-, Dark- und OLED-Mode sowie Zweisprachigkeit (DE/EN).
* **Echtzeit-Daten:** Integration von iCal-Stundenplänen und der OpenMensa API (Mensa-Speiseplan).

## 🛠 Technische Umsetzung
* **Methodik:** Agiles Prototyping in Design-Sprints.
* **Technik:** Umgesetzt mit purem HTML, CSS und JavaScript für maximale Geschwindigkeit.
* **Design-System:** Konsequente Nutzung eines UI-Styleguides (feste Farben, Schriften und Komponenten).
